## Installation

    $ gem build caesar_cipher (eq. caesar_cipher.gemspec)
    $ gem install caesar_cipher (eq. caesar_cipher-0.2.0.gem)

## Usage

In Ruby file:

    require 'caesar_cipher'

In irb:

    $ irb -rubygems
    > require 'caesar_cipher'
